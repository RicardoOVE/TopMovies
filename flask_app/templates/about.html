TopMovies: a Coding Dojo student's project

This project was made as part of the Coding Dojo's web development bootcamp. The site was created using HTML, CSS, js & Python. Bootstrap was used for styling the site, PyMySQL for the DB management & the Flask framework was used for routing and sessions
